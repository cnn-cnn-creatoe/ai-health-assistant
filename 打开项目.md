# 在 Cursor 内置浏览器中打开项目

## 🚀 快速步骤

### 1. 确保服务器运行

服务器应该已经在后台启动。如果还没运行，请：

**后端终端**:
```powershell
cd C:\Users\nan\Desktop\agent\agenttest\backend
.\venv\Scripts\python.exe run.py
```

**前端终端**:
```powershell
cd C:\Users\nan\Desktop\agent\agenttest\frontend
npm run dev
```

### 2. 在 Cursor 内置浏览器中打开

#### 方法 A: 使用命令面板（推荐）

1. **按 `Ctrl+Shift+P`** (Windows/Linux) 或 **`Cmd+Shift+P`** (Mac)
2. **输入**: `Simple Browser` 或 `Browser`
3. **选择**: "Simple Browser: Show" 或 "Open Simple Browser"
4. **在地址栏输入**: `http://localhost:5003`
5. **按 Enter**

#### 方法 B: 使用快捷键

某些 Cursor 版本支持：
- 按 `Ctrl+K` 然后输入 URL
- 或使用其他浏览器快捷键

### 3. 验证访问

打开后应该看到：
- ✅ AI 健康助手的主界面
- ✅ 顶部导航栏（首页、问诊、健康记录、预约）
- ✅ Dashboard 页面内容

## 📋 项目 URL

- **前端应用**: http://localhost:5003
- **后端 API**: http://localhost:8000
- **API 文档**: http://localhost:8000/docs

## 🔍 如果打不开

### 检查服务器状态

在浏览器中访问：
- http://localhost:8000/health (应该返回 `{"status":"healthy"}`)
- http://localhost:5003 (应该看到应用界面)

### 常见问题

1. **服务器未启动**
   - 检查终端窗口是否在运行
   - 查看终端中的错误信息

2. **端口被占用**
   - 检查是否有其他程序占用 5003 或 8000 端口
   - 可以修改端口配置

3. **Cursor 内置浏览器不可用**
   - 使用外部浏览器：直接访问 http://localhost:5003
   - 或使用已打开的测试页面

## 💡 提示

- 服务器启动需要几秒钟时间
- 保持终端窗口打开
- 修改代码后，Vite 会自动刷新页面
- 按 `F12` 打开开发者工具进行调试
