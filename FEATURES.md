# 功能演示说明

本文档详细说明 AI Health Assistant 的各项功能和使用方法。

## 📸 功能截图说明

### 1. 仪表盘（Dashboard）

**位置**：首页 `/dashboard`

**功能**：
- 显示关键健康指标（血压、心率、体温）
- 健康评分可视化
- 7天/30天趋势图表
- 快捷功能入口
- **紧急联系按钮**（右上角红色按钮）

**截图要点**：
- 展示健康数据卡片
- 展示趋势图表
- 展示快捷入口网格
- 突出显示紧急联系按钮

### 2. AI 健康助手（Chat）

**位置**：`/chat`

**功能**：
- 智能症状分析
- 多轮对话支持
- 紧急情况自动检测
- **紧急联系按钮**（页面右上角）

**截图要点**：
- 展示对话界面
- 展示 AI 回复示例
- 展示紧急情况检测提示
- 展示紧急联系按钮

### 3. 健康档案管理（Records）

**位置**：`/records`

**功能**：
- 文件上传（支持 PDF、图片、Office 文档）
- 分类管理（报告/指标/症状/其他）
- 在线预览（图片、PDF）
- 记录删除
- **数据导出**（PDF/JSON）

**截图要点**：
- 展示上传区域和分类选择
- 展示文件列表
- 展示文件预览效果
- 展示导出按钮

### 4. 体检提醒（Checkup Reminders）

**位置**：`/checkup`

**功能**：
- 添加体检提醒
- 设置提醒周期（6个月/1年/自定义）
- 到期提醒通知
- 多提醒管理

**截图要点**：
- 展示添加表单
- 展示提醒列表
- 展示到期提醒标识
- 展示编辑/删除功能

### 5. 紧急联系功能

**位置**：Dashboard 和 Chat 页面

**功能**：
- 一键拨打120急救电话
- 快速联系紧急联系人
- 多场景快速访问

**截图要点**：
- Dashboard 顶部的紧急联系卡片
- Chat 页面右上角的紧急按钮
- 点击后的拨号界面（移动端）

### 6. 数据导出功能

**位置**：健康档案页面

**功能**：
- PDF 报告导出（包含所有健康记录）
- JSON 数据导出（便于备份和迁移）

**截图要点**：
- 导出按钮位置
- PDF 报告预览
- JSON 文件下载

## 📝 功能亮点说明

### 1. 紧急联系功能

**为什么重要**：
- 紧急情况下快速拨号，节省时间
- 多场景快速访问，提高可用性
- 符合医疗应用的安全要求

**使用场景**：
- 用户感到不适时快速拨打120
- AI 检测到紧急情况时引导用户使用
- 紧急联系人快速联系

### 2. 数据导出功能

**为什么重要**：
- 数据备份，防止丢失
- 就医时携带健康报告
- 数据迁移和分享

**使用场景**：
- 定期备份健康数据
- 就医前导出健康报告
- 更换设备时迁移数据

### 3. 体检提醒功能

**为什么重要**：
- 定期体检是健康管理的重要环节
- 自动提醒避免遗忘
- 完整的健康管理闭环

**使用场景**：
- 设置年度体检提醒
- 专项检查提醒
- 定期复查提醒
