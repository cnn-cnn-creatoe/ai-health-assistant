# 提示词工厂 Skill

## 使用场景

当用户需要：
- 创建可复用的提示词模板
- 批量生成相似场景的提示词
- 建立提示词库和知识体系
- 优化和标准化提示词格式
- 为不同 AI 模型适配提示词

## 提示词模板类型

### 1. 代码生成模板
- 函数/类生成
- 测试用例生成
- 代码重构和优化
- 文档和注释生成

### 2. 内容创作模板
- 文章和博客写作
- 营销文案生成
- 产品描述编写
- 社交媒体内容

### 3. 数据分析模板
- 数据提取和清洗
- 数据分析和可视化
- 报告生成
- 洞察总结

### 4. 任务自动化模板
- 工作流自动化
- 邮件和通知生成
- 任务分解和规划
- 决策支持

## 模板结构

### 基础模板

```
# 角色定义
你是一个 [角色]，擅长 [领域]。

# 任务描述
请帮我 [具体任务]。

# 上下文
[相关背景信息]

# 输出要求
- 格式：[指定格式]
- 长度：[指定长度]
- 风格：[指定风格]

# 约束条件
- [约束1]
- [约束2]
```

### 高级模板（带变量）

```
# 角色
你是一个 {{role}}，在 {{domain}} 领域有 {{experience}} 年经验。

# 任务
{{task_description}}

# 输入
{{input_data}}

# 输出格式
{{output_format}}

# 示例
{{example}}
```

## 生成流程

### 1. 需求分析
- 明确使用场景和目标
- 识别关键变量和参数
- 确定输出格式要求

### 2. 模板设计
- 选择合适的模板结构
- 定义变量和占位符
- 设置约束和验证规则

### 3. 模板生成
- 填充具体内容
- 替换变量和参数
- 优化语言和结构

### 4. 测试和优化
- 使用示例输入测试
- 评估输出质量
- 迭代改进模板

## 提示词优化技巧

### 结构化组织
- 使用清晰的标题和分段
- 采用列表和表格提高可读性
- 使用标记和分隔符

### 变量化设计
- 识别可替换的部分
- 使用占位符标记变量
- 提供默认值和选项

### 上下文增强
- 添加相关背景信息
- 提供示例和参考
- 说明预期用途

### 约束明确
- 明确输出格式要求
- 设置长度和风格限制
- 指定禁止或必须的内容

## 模板库管理

### 分类组织
```
prompts/
├── code-generation/
│   ├── function-template.md
│   ├── test-template.md
│   └── refactor-template.md
├── content-creation/
│   ├── blog-post.md

│   └── product-description.md
└── data-analysis/
    ├── extract-template.md
    └── report-template.md
```

### 版本控制
- 记录模板版本和变更
- 维护变更历史
- 标记已弃用的模板

### 使用统计
- 跟踪模板使用频率
- 收集反馈和改进建议
- 优化常用模板

## 最佳实践

- **保持简洁**：避免不必要的冗长描述
- **明确具体**：使用具体的指令而非抽象概念
- **提供示例**：展示期望的输出格式
- **迭代优化**：基于使用反馈持续改进
- **文档化**：记录模板用途和使用方法

## 注意事项

- 根据目标 AI 模型调整提示词风格
- 平衡详细程度和简洁性
- 定期更新和维护模板库
- 测试不同场景下的效果
